# Task 7: Identify and Remove Suspicious Browser Extensions

## Objective
To identify, review, and remove suspicious or unnecessary browser extensions in order to improve browser security and performance.

## Tools Used
- Web browser (Google Chrome)

## Steps Followed
1. Opened the browser extension manager.
2. Reviewed all installed extensions.
3. Checked permissions and user reviews.
4. Identified unused or suspicious extensions.
5. Removed unnecessary extensions.
6. Restarted the browser to check performance.

## Suspicious / Unnecessary Extensions Found
Details are documented in the extensions_review.txt file.

## Security Risks of Malicious Extensions
- Data theft
- Password stealing
- Browser hijacking
- Tracking user activity
- Injecting ads or malware

## Conclusion
Reviewing and managing browser extensions regularly helps prevent security risks and improves browser performance.
