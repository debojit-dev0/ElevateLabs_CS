1. **How can browser extensions pose security risks?**
Browser extensions can access websites and user data. Malicious extensions may steal information, track browsing activity, inject ads, or install malware.

2. **What permissions should raise suspicion?**
Permissions like access to all websites, reading browsing history, modifying web pages, or accessing cookies should raise suspicion.

3. **How to safely install browser extensions?**
Install extensions only from official browser stores, check reviews and ratings, review permissions, and avoid unknown developers.

4. **What is extension sandboxing?**
Extension sandboxing limits what an extension can access so that even if it is malicious, the damage is reduced.

5. **Can extensions steal passwords?**
Yes. If an extension has permission to read website data or cookies, it can potentially steal passwords.

6. **How to update extensions securely?**
Enable automatic updates from the official browser store to ensure security fixes are applied.

7. **Difference between extensions and plugins?**
Extensions add features to the browser, while plugins run external content like media or software inside the browser.
