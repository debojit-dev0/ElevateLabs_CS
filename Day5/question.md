1. **What is Wireshark used for?**
To capture and analyze network packets.

2. **What is a packet?**
A small unit of data sent over a network.

3. **How to filter packets in Wireshark?**
By using display filters like tcp, udp, or dns.

4. **TCP vs UDP?**
TCP is reliable and connection-based. UDP is faster but connectionless.

5. **What is a DNS query packet?**
It resolves a domain name to an IP address.

6. **How does packet capture help troubleshooting?**
It shows where communication is failing.

7. **What is a protocol?**
A set of rules for network communication.

8. **Can Wireshark decrypt encrypted traffic?**
Only if encryption keys are available.
