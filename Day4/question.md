1. **What is a firewall?**
A firewall is a security system that controls network traffic based on rules.

2. **Stateful vs Stateless firewall?**
Stateful tracks active connections. Stateless checks each packet individually.

3. **Inbound and outbound rules?**
Inbound controls incoming traffic. Outbound controls outgoing traffic.

4. **How does UFW simplify firewall management?**
It provides easy commands instead of complex firewall rules.

5. **Why block port 23 (Telnet)?**
Telnet sends data in plain text and is insecure.

6. **Common firewall mistakes?**
Blocking required ports, weak rules, or forgetting to update rules.

7. **How does a firewall improve security?**
It prevents unauthorized access to the system.

8. **What is NAT in firewalls?**
NAT hides internal IP addresses by mapping them to a public IP.
